# AI-Fusang v1.9 节点问题角标 + 多问题列表 实现说明

## 修改内容

### 1. `getWorstQualityIssue()` → `getQualityIssues()`
- 原来只返回**最严重的一个问题**
- 现在返回**所有问题的排序数组**（high > medium > low）
- 保留旧函数名做兼容（内部调用新函数取 `issues[0]`）

### 2. 节点角标逻辑（`drawNodes()` 内）
- **1 个问题**：右上角显示类型图标（🔴🟠🟡🟢🔵⚪）
- **2+ 个问题**：右上角显示红色数字圆标（如 `3`）

### 3. InfoCard 多问题列表（`drawInfoCard()` 内）
- 显示**前 3 个问题**（按严重程度排序）
- 第 3 个后面显示 `...还有 N 个`（如果 >3）
- 每个问题一行：`图标 + 详细数据`
- InfoCard 高度**自适应**（多问题节点自动扩展）

### 4. 数据结构
```json
"quality": {
  "score": 45,
  "issues": [
    {"type": "highCoupling", "severity": "high", "detail": "模块间依赖 12 个"},
    {"type": "deepNesting", "severity": "medium", "detail": "最大深度 5 层"}
  ]
}
```

## 验证步骤
1. 刷新 `http://localhost:8081`
2. 点「🔬 探伤」进入探伤模式
3. 演示播放完成后，点击**红色节点** → InfoCard 应显示：
   - 质量评分（45/100）
   - 问题列表（前 3 个，按严重程度排序）
   - 详细指标（耦合/嵌套/行数/复杂度）
   - 代码位置 + 修复建议
   - 「📋 查看报告」+「📍 定位」按钮
4. 如果节点有 2+ 个问题 → 节点右上角显示红色数字圆标
5. 如果节点只有 1 个问题 → 节点右上角显示类型图标
